# 创建 Python 接口测试

## 目标

- 验证 HTTP 路由和响应结构
- 验证参数校验和错误语义
- 验证鉴权、权限与回归场景

## 适用场景

- 新增或修改 FastAPI 路由
- 修复接口缺陷
- 需要做本地或测试环境回归

## 最小用例清单

- 成功路径
- 参数校验失败
- 未登录或越权
- 业务错误
- 缺陷回归

## 推荐落点

- 仓库内测试：`tests/api/` 或既有 API 测试目录
- 运行方式：`pytest`

## 输出格式

```markdown
## 接口测试结果
- 目标模块：
- 接口清单：
- 测试文件：
- 执行命令：
- 最终结果：PASS / FAIL
- 风险与待补充：
```
