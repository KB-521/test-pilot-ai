# 记录经验

沉淀有价值的经验供未来参考。

## 分类

| 标签         | 用途         |
| ------------ | ------------ |
| `[BUG]`      | 缺陷解决方案 |
| `[PATTERN]`  | 成功模式     |
| `[PITFALL]`  | 需要避免的坑 |
| `[DECISION]` | 重要决策     |
| `[PERF]`     | 性能优化     |

## 模板

```markdown
# [分类] 标题

## 摘要
[一句话]

## 问题
[问题是什么]

## 解决方案
[如何解决]

## 代码示例
```language
// 修改前（不好）
...

// 修改后（更好）
...
```

## 预防
[未来如何避免]

## 标签
#tag1 #tag2
```

## 示例

```markdown
# [BUG] 用户查询空指针

## 摘要
在访问用户属性前进行空值判断。

## 问题
当用户不存在时 API 崩溃。

## 解决方案
增加空值判断并返回 404。

## 预防
- 为空值场景添加单元测试
- 使用 Optional 类型
```
