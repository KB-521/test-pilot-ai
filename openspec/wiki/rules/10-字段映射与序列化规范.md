# 字段映射与序列化规范

## 文档说明

- 覆盖规则：Pydantic 字段命名、别名、序列化、反序列化和领域字段映射规则。
- 核心要求：外部 API 字段、内部领域字段和持久化字段的映射必须显式，不靠隐式猜测。
- 应用场景：定义 schema、DTO、ORM 模型和前后端契约。
- 命中主题：alias、serialization、field mapping
- 触发关键词：Pydantic alias,序列化,字段映射,DTO,ORM
- 关联 skill：python/backend-design、python/req-impl、web/task-impl
- 前置规则：06、07
- 后置规则：21、22
- 排除场景：纯页面样式调整

## 规则

1. 对外字段名以 API 契约为准。
2. 内部字段命名与数据库字段命名不一致时，要有显式映射层。
3. 不要把 ORM 模型直接作为对外响应模型。
4. 字段默认值和可空语义要在 schema 中写清。
