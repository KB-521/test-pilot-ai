## 何时使用

- 需求或实现闭环后。
- 解决复杂问题后。
- 用户明确要求“记住这个”时。

- `openspec/wiki/business/INDEX.md`

## 工作流

```text
识别经验 -> 分类 -> 记录 -> 更新索引 -> 关联到需求或设计
```

## 分类

| 分类 | 用途 |
| --- | --- |
| `[BUG]` | 缺陷解决方案 |
| `[PATTERN]` | 可复用模式 |
| `[PITFALL]` | 需要避免的坑 |
| `[DECISION]` | 设计决策 |
| `[TOOL]` | 工具与脚本技巧 |

## 使用的技能

- `common/experience-record`

## 自动沉淀触发条件

- 需求闭环
- 修复棘手缺陷
- 新增一条稳定规则
- 用户明确要求“记住这个”
